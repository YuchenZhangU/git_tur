## Init Repo

```bash
git init # initialize a repo

git status # check current working status

# working flow
## add
git add <filename>
git add .

## commit
git commit -m "commit message"


# switch between commits (return to a time snapshot)
git checkout <commitid> # example git checkout effe34
git checkout <branchname> # example git checkout master

# banching
git branch <branchname> # attach a branchname to current commit
```
